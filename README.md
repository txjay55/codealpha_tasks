# Iris Classification using Logistic Regression

## 📌 Overview
This project classifies iris flowers into three species:
- Setosa
- Versicolor
- Virginica

## ⚙️ Model
- Logistic Regression
- Scikit-learn

## 📊 Results
- Accuracy: 100%
- Cross-validation score: ~99%

## 📈 Features Used
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 📉 Visualization
Includes decision boundary plot using petal features.

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
